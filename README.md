# Spring Boot 익명게시판

## 프로젝트 정보
| **내용**              | **설명**                          |
|---------------------|----------------------------------|
| **프로젝트 이름**     | 게시판 구현 프로젝트 (Board Application) |
| **Spring Boot 버전**   | 3.4.1                            |
| **JDK 버전**          | 17                               |
| **사용한 데이터베이스** | H2 (개발용), MySQL (실제 환경)        |

| **빌드 도구**         | Gradle                           |
